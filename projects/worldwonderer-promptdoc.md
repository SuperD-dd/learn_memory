# promptdoc

> 该项目用于管理多版本、多场景、多适用模型的 Prompt 模板集合。基于 Python 和 Flask 框架开发，提供了一组 RESTful API 和管理后台，用于对 Prompt 模板进行创建、检索、更新和删除等操作。
