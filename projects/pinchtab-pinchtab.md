# pinchtab

> PinchTab 是一个基于 Go 开发的浏览器自动化控制服务，能够通过 CLI 或 HTTP API 让 AI 代理直接操控 Chrome，支持多实例隔离、无头运行、文本快照与隐身增强等能力，旨在以更低 token 成本完成网页交互与数据提取，适合 Linux、macOS 与 Docker 环境使用。
