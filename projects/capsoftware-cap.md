# Cap

> 一款简单易用的分享和录制屏幕视频工具，目前提供了 Web 和 macOS 客户端。它免费、开箱即用，能够轻松录制屏幕视频并生成链接直接分享，任何人点开链接就能观看视频，极大地简化了录屏分享流程，可当作 Loom 的开源替代品。
