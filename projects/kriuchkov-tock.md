# tock

> 一个基于 Go 语言和 Clean Architecture（清洁架构）原则开发的命令行时间追踪工具，提供美观的交互式终端 UI（TUI）日历视图，支持以纯文本格式记录活动，并能与 TimeWarrior 数据兼容。
