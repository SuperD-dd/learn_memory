# Proma

> Proma 是一个基于 Claude Agent SDK、Electron 和 React 开发的开源 AI 桌面应用，能够提供多模型对话、通用 Agent、Agent Teams、记忆与远程飞书机器人协作能力，旨在让用户以本地优先的方式高效完成复杂任务，并支持多供应商灵活接入。
