# defuddle

> 一个基于 TypeScript 开发的网页内容提取工具，能够智能识别并移除网页中的评论、侧边栏、页眉页脚等非必要元素，仅保留主要内容，旨在为 HTML 转 Markdown 转换器提供更干净、一致的输入，支持浏览器和 Node. Js 环境使用。
