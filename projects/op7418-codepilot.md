# CodePilot

> CodePilot 是一个基于 Electron、Next. Js 与 Claude Agent SDK 开发的桌面 GUI 客户端，能够通过可视化界面完成对话编码、项目管理与上下文查看，旨在降低 Claude Code 的使用门槛，提升开发效率，支持 macOS、Windows 和 Linux 平台使用。
