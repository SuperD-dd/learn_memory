# FList

> 一个很有想法的有意思的项目。可以将 GitHub Releases,Hugging Face,文件 url 等，渲染成类似网盘的文件列表显示在网页上，方便用户下载开源软件。支持视频、音频、图片、PDF 等文件的在线预览。
