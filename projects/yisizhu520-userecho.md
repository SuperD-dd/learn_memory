# userecho

> 一个基于 FastAPI 和 Vue 3 开发的开源用户反馈管理与洞察平台，能够自动合并重复反馈、智能识别截图内容并生成专业回复，旨在帮助产品团队高效处理用户声音并洞察需求痛点，支持 Docker 部署和自托管使用。
