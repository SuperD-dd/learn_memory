# lynx-proxy

> 一款基于 Rust 语言开发的代理抓包工具，项目采用 hyper、axum、tower 等主流 Rust 网络库，以满足不同在开发阶段的需求，比如移动端开发时候查看接口，脚本注入，web 端开发时候将静态资源指向本地服务。
