# video-helper

> 一个基于 Python FastAPI 和 Next. Js 开发的智能视频学习助手，能够自动分析视频内容并生成思维导图、章节摘要和学习笔记，旨在大幅提升视频学习与知识复盘的效率，支持 Windows、macOS 和 Linux 跨平台使用。项目集成了 Whisper 语音转录和 LLM 智能分析能力，实现了思维导图与内容的双向联动交互，帮助用户快速掌握视频核心知识。
