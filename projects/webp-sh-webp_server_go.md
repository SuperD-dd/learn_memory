# webp_server_go

> 一个用 Golang 编写的轻量级服务器，它可以根据我们自己的需求，动态地将各种格式的图片转换为 WebP 格式，从而提高网页加载速度和浏览体验。目前支持的图像格式包括 JPEG、PNG、BMP 和 GIF（仅静态图像）。
