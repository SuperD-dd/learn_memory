# pixzip

> 一款快速、免费的图片压缩软件，支持 JPG / PNG / WebP / AVIF 等主流格式，支持 Windows 10+，macOS 11+ 操作系统。 此前周刊还推荐过类似的软件，有：ImageOptim (opens new window)，TinyPNG4Mac (opens new window)，Imagine (opens new window)。我一直在用 Imagine。
