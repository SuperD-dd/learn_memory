# http-relay

> 一个基于 Go 语言开发的轻量级 HTTP 中继工具，能够将本地 HTTP 请求转发至路径中指定的绝对 URL 并原样返回上游响应，旨在为开发者提供便捷的请求转发与调试能力，支持上游代理配置、流量抓包 dump 及敏感认证头掩码功能，适用于本地开发测试与网络调试场景，可跨平台使用。
