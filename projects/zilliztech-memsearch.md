# memsearch

> memsearch 是一个基于 Python 开发的 Markdown 优先记忆系统，能够为 AI Agent 提供持久化记忆、语义检索与自动索引能力，采用 Markdown 作为事实来源，结合向量搜索、内容去重和实时同步机制，旨在帮助开发者低成本构建可追溯、可扩展、可本地运行的智能体记忆层。
