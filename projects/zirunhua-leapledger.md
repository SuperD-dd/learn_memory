# LeapLedger

> 一个的前后端分离免费开源的记账软件，服务端使用 Gin 框架，基于 Nats、Mysql、Redis 实现，带来快速的响应和稳定的服务。使用 docker 即可快速部署和构建客户端安装包。客户端基于 flutter 构建。
