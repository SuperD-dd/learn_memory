# djump

> 一个基于 Go 开发的跨平台内网 Web 服务代理工具，能够通过可视化界面统一管理和暴露内网站点，支持 HTTP/HTTPS 反向代理、Token 与 Session 认证及自定义 SSL 证书，旨在以更低门槛实现家庭或办公场景下的安全远程访问。
