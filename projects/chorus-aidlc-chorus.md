# Chorus

> Chorus 是一个基于 Next. Js、TypeScript、PostgreSQL 与 MCP 构建的 AI Agent 协作平台，能够管理会话生命周期、任务状态、子 Agent 编排与人机审核流程，旨在把从需求分析到开发交付的全过程标准化、可观测化和可恢复化，支持本地部署、Docker 与 AWS 环境使用。
