# openclaw-kasmvnc

> 一个基于 Docker、KasmVNC 与 OpenClaw 打造的可视化部署项目，能够一键提供浏览器远程桌面与 OpenClaw 网关运行环境，支持在容器内完成安装、升级、重启等完整生命周期管理，旨在降低部署和运维门槛，适用于 Windows、macOS 和 Linux 平台使用。
