# cc-wf-studio

> 一个专为 Claude Code 设计的开源可视化工作流编辑器（VS Code 插件）。它允许开发者通过直观的拖拽界面来设计复杂的 AI 智能体工作流、自定义斜杠命令（Slash Commands）以及集成 MCP（Model Context Protocol）工具。
