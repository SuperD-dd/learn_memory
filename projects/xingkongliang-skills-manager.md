# skills-manager

> 一个基于 React、TypeScript、Tauri 2 与 Rust 开发的桌面技能管理工具，能够统一管理并同步 AI agent skills 到 Cursor、Claude Code、Copilot 等多种编码工具，支持场景切换、更新追踪与文档预览，旨在提升多工具协作下的技能复用效率与管理一致性。
