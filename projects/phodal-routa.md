# routa

> Routa 是一个基于 TypeScript、Next. Js 与 Rust 构建的多智能体协作开发平台，能够将自然语言需求解析为结构化任务，并通过 MCP、ACP、A 2 A 协调 Claude Code、OpenCode、Gemini 等智能体协同工作，旨在提升复杂研发任务的拆解、执行与验证效率，支持桌面端与 Web 演示使用。
