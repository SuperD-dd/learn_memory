# PUAClaw

> PUAClaw 是一个基于 Markdown 构建的提示词说服技术研究项目，专注于整理和分类面向大语言模型的非常规提示策略，具有多语言文档、体系化分类和案例化说明等特点，核心价值在于为研究者提供一个兼具梳理、分析与娱乐表达的开源参考框架。
