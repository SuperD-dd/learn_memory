# markstream-vue

> 一个基于 Vue 3 开发的流式 Markdown 渲染器，专为 AI 对话场景设计，能够在 SSE/WebSocket 流式数据传输中实现零闪烁的实时渲染，支持 Monaco 编辑器增量更新、Mermaid 图表渐进渲染和 KaTeX 公式快速计算，采用虚拟滚动和增量批处理技术确保大文档的流畅体验，支持 Vue 2 和 React 框架使用。
