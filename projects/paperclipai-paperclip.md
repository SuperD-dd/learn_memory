# paperclip

> 一个基于 TypeScript 和 React 开发的 AI 代理编排平台，能够协调多个 AI 代理团队自主运营业务，旨在帮助用户构建零人工运营的公司，支持多公司隔离、预算控制、目标对齐和治理审批等功能，可通过移动端随时随地管理自主业务。
