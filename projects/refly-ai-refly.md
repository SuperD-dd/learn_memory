# refly

> Refly 是一个面向 Agent Skills 构建与交付的开源平台，能够将企业 SOP 和业务流程快速编译为可执行工作流与标准化技能，支持 API、Webhook、Claude Code 等多种运行方式，具备可视化编排、版本化管理和可控执行等特点，旨在帮助团队高效构建稳定、可复用、可部署的智能体基础设施。
