# nps

> 一款轻量级、高性能、功能强大的内网穿透代理服务器。支持 tcp、udp、socks 5、http 等几乎所有流量转发，可用来访问内网网站、本地支付接口调试、ssh 访问、远程桌面，内网 dns 解析、内网 socks 5 代理等等，并带有功能强大的 web 管理端。
