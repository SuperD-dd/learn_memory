# neo

> 一个基于 TypeScript 开发的 Chrome 扩展工具，能够将任意 Web 应用转换为可调用的 API。通过被动捕获浏览器流量、自动生成 API Schema，让 AI Agent 能够直接重放 API 调用或通过无障碍树进行 UI 自动化操作。无需官方 API，支持 Chrome 和 Electron 应用，适用于跨平台自动化场景。
