# nyaterm

> 一个基于 Tauri、React 和 Rust 开发的现代远程终端工作空间，能够集成 SSH、本地 Shell、Telnet 和 Serial 会话，提供 SFTP 文件传输、AI 助手及加密同步功能，旨在提升多终端操作效率，支持 Windows、macOS 和 Linux 平台使用。
