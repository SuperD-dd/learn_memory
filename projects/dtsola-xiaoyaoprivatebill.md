# xiaoyaoprivatebill

> 一个基于 Vue 3、Flask 与 Pandas 开发的个人账单分析工具，能够一键解析支付宝和微信账单，并通过可视化方式展示收支、分类与消费趋势，旨在帮助用户在本地安全看清每一分钱的去向，支持 Docker 部署及桌面、移动端浏览器使用。
