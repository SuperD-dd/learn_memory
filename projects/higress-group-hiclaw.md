# hiclaw

> HiClaw 是一个基于 OpenClaw 与 Matrix 构建的开源 Agent Teams 系统，能够让 Manager 统一协调多个 Worker 完成协作任务，并通过 IM 实现全程可视化与人工介入，旨在降低多智能体部署和管理门槛，支持 Linux、macOS、Windows 等平台使用。
