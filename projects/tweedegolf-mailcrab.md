# mailcrab

> 一个由 Rust 编写的轻量级开发测试用邮件服务器，其灵感源自 MailHog 和 MailCatcher。它旨在为开发环境提供一个“只进不出”的 SMTP 接收器，开发者可以通过其内置的 Web 界面实时查看、检查发出的邮件详情（包括 HTML 渲染、附件、Header 等），而无需担心邮件被真实发送给用户。
