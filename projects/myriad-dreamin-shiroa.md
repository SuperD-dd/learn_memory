# shiroa

> 一个基于 Typst 与 Rust 开发的在线书籍生成工具，能够将纯 Typst 内容构建为现代化网页书籍，适合技术文档与教程发布。它强调纯 Typst 工作流、主题扩展与多平台构建能力，核心价值在于用更简洁一致的方式提升文档编写与发布效率。
