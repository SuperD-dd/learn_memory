# CronPilot

> CronPilot 是一个基于 Tauri v 2、React 与 Rust 开发的本地 crontab 可视化管理桌面应用，能够用图形界面创建、编辑、执行和同步定时任务，支持日志查看、冲突检测与中英双语，旨在降低定时任务管理门槛并提升运维效率，适合 macOS 与 Linux 平台使用。
