# OpenClaw-bot-review

> 一个基于 Next. Js 与 TypeScript 开发的 OpenClaw 可视化仪表盘，能够集中查看机器人、模型、会话与平台连通状态，直接读取本地配置和会话文件，无需数据库，旨在提升多 Agent 场景下的运维效率与监控体验，并支持中英文界面和 Docker 部署。
